# env-setup

開発環境の構築手順（OS・プログラム実行環境）を管理するリポジトリです。

## 概要

本リポジトリは、開発に必要なOSの初期設定および各言語の実行環境を構築するための手順を記載した備忘録です。
不特定多数の方が参照することを想定し、汎用的な構成を目指しています。
